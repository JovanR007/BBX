# BeyBracket (BBX)

A Beyblade Tournament Management System.

## Project Structure

- `web/`: Next.js web application for the tournament interface.
- Root scripts (`*.js`): Node.js scripts for backend logic and database management.

## Getting Started

### Web Application

1. Navigate to the `web` directory:
   ```bash
   cd web
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

### Backend Scripts

Run the scripts using Node.js, for example:
```bash
node check_top_cut.js
```
